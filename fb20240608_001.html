【加速器。路跑】

以前在清華，大一體育必修課，會被要求參加環校路跑；

大三開始，同學們都宅宅不打桌球了，只得自己去跑步，

跑跑便跑出一些興趣，奔跑在十八尖山，自放山水之間。

來加州的第二個月，參加醫學院舉辦的PH run，

報名費45鎂，用來資助pulmonary hypertension

#肺動脈高壓

校園一隅跑個3.1mile

當時尚未適應乾冷天氣的我，穿著長褲外套，

是三十人先鋒部隊中唯一的熱帶島嶼冬衣者。

28/629

回顧清大路跑的校友組

8/386

可知美國慢跑文化的盛行

(清大是給錢還不一定參加，美國是要給錢才能參加呢!)

\\

第五個月，參加慢跑社舉辦的Dish run，

圍繞著史丹佛大耳朵(Stanford dish)跑3.25mile，

報名費也要價35鎂。

終於換上短袖短褲，

選擇了"For fun組"(慢跑社朋友都在競賽組....)

也在抵達終點前使用Naruto run帥氣飛越，

跟應該是日本人的社團朋友擊掌。

7/164

而且第六名的，同樣是上次推著嬰兒車的大叔，

他總是在我跑到一半時，讓嬰兒『超車』眾人。

\\

第八個月，終於，終於有加速器實驗室的路跑，

更是因為疫情後睽違四年，才舉辦的49th路跑。

這場路跑對我而言的意義之重大，

在於當我第一天在入職實驗室，

第二天便收到信，加速器實驗室的主任，

從台灣人Kao換成另一個美國人John。

而在兩個月前，John從兩千位員工，

隨便挑了十位來開會，問問大家近況。

所有人都一致抱怨實驗室的餐廳又貴又難吃，

我還很囂張地說，自己常常坐交通車回學校，

學校餐廳好吃多了呢！

大老一直笑，我趕緊補充：

「嘿！John，我日前才參加過醫學院和慢跑社舉辦的路跑，

咱們身為加速器實驗室，更該辦一場吧？

我上次跑步，對於加速器物理的電子團運動行為，有深刻的洞見呢！

一團人（a bunch of people）會隨著自身與外界的梯度場，

分成一小團一小團(Micro-bunching)

」

眾人聽到我這小孩子用加速器物理的經典現象唬爛，

也是鼓掌叫好。(有幾位是機械師或人資，可能比較霧煞煞)

大老說：

「其實疫情前都有舉辦，可那個熱衷辦活動的傢伙，已經跑到華盛頓DC了。

Well…..我們還是先把餐廳搞好吧！」

當天回去，我忽然想到，醫學院舉辦的路跑，是為了研究肺動脈高壓的疾病而募款，那咱們既然有餐飲議題，何不故技重施？

趕緊寫信給John:

“What if we reached out to the original organizer of the "slac run" to propose hosting a run for fundraising purposes? We could gather insights and use the funds to transform the slac cafe into an even better space.

(Stanford medical school has hold PH run for 20 years with the similar purpose! )

Best,

yuting

”

其實這類募款的idea，是我更早就跟"當時還不會亂發脾氣的"博後分享，

因為他開車載我繞著長達三公里的直線加速器，並抱怨：

「你知道嗎？這個大門走出去，隨便一家矽谷公司的年薪都是咱老闆苦幹活幹的三倍以上！我們搞科學，搞這些『破事兒』!」

我就說：

「既然研究單位資金不足，何不來搞個募款？邀請祖克伯那些矽谷大老，一起來跑步。」

我便說明各種理由，讓科技大老們有動機來加速器實驗室，加速資金流動。

博後只是笑我說：

「別給我搞事，這裡是國家實驗室，閒雜人等不得進入！」

我終究還是上書大老，大老則回應：

“Yuting,

Thanks for following up and for engaging in the conversation.

Hadn’t heard about the PH run – very impressive. I’ll pull the string on whether/how we’ve used the slac run for broader purposes. Helping the cafeteria is a novel idea, although there may be more ‘worthy’ causes.

”

這個用字遣詞，實在讀不太懂，尤其那個’worthy’，

不過，成功上書的感覺實在不賴！

神奇的是，路跑活動還真的再次啟動，雖然我不知道自己的上書，幫了多少忙，

#報名費15鎂還送超帥加速T

而我也在跟加速器一起奔跑得過程中，

領悟到上週聽到的IBS(Intra-Beam Scattering)理論精隨，

在使用Small angle approximation(或High energy approximation)的情況下，

把動能和位能調控得宜不互相碰撞地穿梭於人群，

早上九點，

寒風刺骨，

穿著外套與長褲的，

恐怕僅有年過花甲以上的組別，

因為開跑沒多久後，我成了前鋒三十人部隊中，

唯一外套長褲、貼著55號牌子的『跑者』，

每次只要經過路邊的加油群眾，

我肯定雙手比五，大喊:『Five Five!!』

路人看到也熱情地回應，

我再把雙手舉高說:『I surrender!!』#我投降

繼續在一群短褲背心的跑著群中穿梭，

\

路線已經過了約兩公里後，

經過一堆三角錐，

我使出精神加持法#自欺精神法

每用手按壓一次三角錐，

便覺得吸取大地精華，能量回升!

忽然感覺自己能衝鋒，往前超越!

直接雙手後擺，來一個Naruto run飛奔!

粗估，最前方不到二十人!

在一個轉彎後，穿越馬路，

我驕傲地把雙手舉高，示意我是55號，

看完車上駕駛狐疑的眼神後，

心滿意足地穿越了馬路。

\

前方忽然看不到任何人?

一回頭，才知道剛剛根本不用轉彎，

一個女生趕緊指著正確的方向.........

來回間，除了約25秒的耗費，

更是讓我的體力與精神大受打擊，

後半場的我，幾乎都把雙手高舉，徹底被自己打敗。
