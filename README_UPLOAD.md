# 真正整合版：Facebook → 原網站文章頁

這版不再新增 Facebook 專區。

做法：
1. 每則 Facebook 貼文都轉成獨立文章頁：`fbYYYYMMDD_###.html`
2. 文章頁格式模仿你原本的 `d20240314.html`
3. 再把文章連結插入既有分類頁，例如 `g2.html`、`g3.html`、`History.html`、`page2.html`
4. 上傳時會覆蓋同名分類頁，點原本分類頁就會看到「Facebook 匯入文章」
